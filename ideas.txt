Программа минимум - приложение, которое из командной строки ищет информацию для файлов из указанной папки и записывает полученные данные в файл.

1. создать html страницу со всеми результатами обработки выбранной папки
2. по названию статьи подбирать схожие статьи и перенаправлять на сайт издательства
3. есть только название статьи, но нет pdf файла - можно найти библиографические данные / отправить на сайт издательства для скачивания
4. получить из cтатьи библиографические данные статей, на которые она ссылается (references)     
5. добавить в html страницу ссылки на статьи из references
6. сделать это как плагин для редактора ТеХа (???)
7. расширить область применения на статьи из других областей знаний (web of science по подписке), поиск через scholar.google 
8. при отсутствии доступа к интернету / ресурсу с библиографическими данными пытаться все библиографические сведения получить оффлайн
9. Утилита командной строки будет иметь систаксис наподобие следующего:
biblio dir_name file_name.bib
При этом будет произведён поиск библиографической информации для всех файлов формата pdf из указанной директории, 
результат будет записан в соответствующем формате в файл file_name.bib.
Можно добавить возможность указания имён документов в виде перечисления, шаблона, тогда перед именем выходного файла нужно будем добавлять специальный идентификатор.
Действия утилиты по шагам:
1. Создаём (или не создаём, если указан уже существующий) выходной файл.
Файлы обрабатываем поочерёдно, подгружая их по надобности. Следующие шаги выполняем для каждого файла.
2. [этот шаг вероятно впоследствии разобъётся на подшаги] 
Из текущего файла извлекаем текстовые данные. Необходимо извлечь название, при возможности ещё авторов
(при нахождении статьи с этим названием убеждаемся в корректности поиска по авторам).
3. Осуществление поискового запроса по названию.
4. Выделение из ответов корректного.
5. Получение информации по ответу и запись её в поля соответсвующей вивлиографической записи в выходном файле.
Переход к следующему файлу или завершение работы.

Замечание: получение поисковой информации возможно различными способами. Так, сайт dblp.org позволяет экспортировать 
результаты в форматах XML, JSON, JSONP, dblp keys (первые 1000) и BibTeX (первые 30). 
Также информацию о каждом найденном документе можно получить в различных форматах, в частности, в формате BibTeX 
(condenced, standart, with crossref view). Нужно проанализировать, что из этого (или других вариантов) удобнее.

3. Загрузка данных из dblp по названию и авторам
3. Генерация bib файла
